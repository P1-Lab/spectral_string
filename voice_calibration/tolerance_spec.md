# Voice Tolerance Specification

A Voice is valid if:

## Spectral tolerance
- frequency deviation < ±1.5%

## Temporal tolerance
- attack deviation < ±10%
- decay deviation < ±12%

## Coupling tolerance
- inter-node variance < threshold per Voice class

---

## Failure Modes

- detuned Voice drift
- mechanical asymmetry
- damping overcorrection
- enclosure resonance mismatch
