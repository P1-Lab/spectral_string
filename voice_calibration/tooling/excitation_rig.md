# Excitation Rig Specification

## Purpose

The excitation rig provides a controlled, repeatable mechanical input system for voice calibration and validation of Spectral String instruments.

It replaces human variability with deterministic strike profiles.

---

## 1. Actuation System

### Primary Mechanism
- Solenoid actuator OR calibrated mechanical striker
- Stroke-controlled linear impactor
- Adjustable force profile (soft → hard excitation range)

### Requirements
- Repeatable strike force deviation: < ±2%
- Mechanical alignment stability: < 0.5 mm drift
- Contact surface hardness standardized across tests

---

## 2. Velocity Profiles

The system must support defined excitation curves:

- Impulse strike (sharp transient)
- Linear ramp strike (controlled energy injection)
- Sustained excitation (for resonance analysis)

Each profile must be reproducible across calibration cycles.

---

## 3. Positional Control

- XY mount or fixed rail system
- Target alignment tolerance: < 0.5 mm
- Z-axis strike depth calibrated per Voice profile

---

## 4. Output Constraint

The rig must not introduce:

- lateral shear forces
- rotational torque on string plane
- uncontrolled secondary contact events

---

## Summary

The excitation rig defines the *input truth condition* for all Voice calibration systems.
