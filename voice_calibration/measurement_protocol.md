# Measurement Protocol

## Purpose
Defines standardized measurement procedure for validating voice states.

---

## 1. Excitation Method

- Solenoid actuator strike system
- Velocity-controlled impulse profile
- Repeatability tolerance: <0.5 mm positional variance

---

## 2. Measurement Stack

- Piezo reference capsule (primary node)
- Tri-axis accelerometer (structural response)
- Laser displacement sensor (optional LDV-class validation)

---

## 3. Recording Conditions

- Controlled room environment
- Fixed mounting rig
- No operator contact during measurement

---

## 4. Output Metrics

- frequency response curve
- decay envelope (time domain)
- harmonic stability index
- inter-node coherence

---

## 5. Validation Rule

If deviation exceeds threshold curves:
→ voice state is rejected
