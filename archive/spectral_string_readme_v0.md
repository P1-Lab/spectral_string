# Spectral String

**Deterministic acoustic instruments for continuous excitation and physically grounded spectral behavior.**

---

## Overview

Spectral String is a family of single- and dual-string instruments that function as a **physically complete acoustic system**.

### Acoustic Independence Principle

Spectral String is a fully playable acoustic instrument.

All sound is generated through:
- mechanical string excitation  
- body resonance  
- structural coupling  

The electronic system is optional and functions only as:
- spectral analysis layer  
- tuning reference system  
- gesture interpretation engine  

If all electronics are removed, the instrument remains fully playable and musically complete.

---

Unlike MIDI controllers or sample-trigger systems, Spectral String produces sound through continuous physical interaction with a tensioned string system.

The performer excites a real string; the instrument resolves that excitation through its mechanical structure into a stable harmonic response field.

Electronics extend interpretation—they do not define sound production.

---

## Instrument Models

### Monos (1-String)

- Single continuous excitation domain  
- Fretless spatial pitch control  
- Maximum sensitivity to micro-gesture input  

---

### Duos (2-String)

- Dual independent excitation channels  
- Interval structures derived from deterministic proportional systems (including φ-based relationships)  
- Cross-string harmonic interaction as a structural feature, not layering  

---

## Core Principles

### 1. Continuous Excitation

All sound emerges from continuous physical interaction:

- plucking  
- striking  
- sliding  
- pressure modulation  
- optional bowing  

There are no discrete trigger zones.

All interaction is spatially continuous and mechanically grounded.

---

### 2. Deterministic Spectral Behavior

Pitch and harmonic behavior are determined by physical parameters:

- string length  
- tension state  
- excitation position  
- structural coupling ratios  

The system does not interpret input.

It resolves physical conditions into spectral outcomes.

---

### 3. Acoustic Authority

The instrument is defined first as a physical object:

- fully playable without power  
- mechanically stable under dynamic excitation  
- tactile response is the primary feedback system  

Electronics are an extension layer, not a requirement for sound generation.

---

## Tuning System

A mechanical lever enables instantaneous switching between reference tunings:

- **A = 440 Hz** (modern ensemble standard)  
- **A = 415 Hz** (historical / solo reference)

This operates as a **global spectral reinterpretation state**, not a physical retuning of the string.

### Result:

- no re-tensioning required  
- no interruption of performance  
- immediate harmonic frame shift  

---

## Performance Modes

### Planar Mode

- instrument referenced to a stable horizontal plane  
- high precision positional control  
- fine-grained articulation behavior  

---

### Vector Mode

- instrument orientation influences response  
- gravity affects damping and energy distribution  
- emphasis on force direction over surface mapping

## Relationship to CajonBlocks

Spectral String and CajonBlocks share a common underlying model:

**Nodal Response Engine (NRE)**

- CajonBlocks: discrete spatial excitation nodes  
- Spectral String: continuous linear excitation domain  

Both systems implement the same physical-to-spectral resolution principles, expressed through different geometries.
