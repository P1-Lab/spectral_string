# Spectral String — Instrument Constitution

Spectral String is a physically grounded acoustic instrument system with optional electronic interpretation layers.

It is not a controller, not a synthesizer, and not a sound model.

It is a mechanically complete instrument family.

---

## 1. Definition of the Instrument

Spectral String is defined as:

a tensioned string system coupled to a resonant body that produces sound through mechanical excitation alone.

Sound exists before electronics.

---

## 2. Acoustic Primacy Rule

The instrument must function fully without power.

This means:

- strings produce audible sound on their own
- resonance is structurally sufficient for musical output
- performance remains valid with zero electronics attached

Electronics do not define sound.

---

## 3. What Electronics Are (Optional Layer)

If present, electronics only perform:

- sensing
- measurement
- interpretation
- state modeling (NRE)

They do not:

- generate audio
- define pitch
- replace acoustic behavior

---

## 4. Mechanical System

The instrument is composed of:

- tensioned string(s)
- bridge and nut boundary system
- resonant body (wood/composite)
- human excitation interface (pluck, slide, strike, bow optional)

Sound is the result of physical energy transfer through this structure.

---

## 5. Nodal Response Engine (NRE)

The NRE is an observational system.

It models:

- excitation position
- velocity of interaction
- damping conditions
- harmonic response behavior

It does NOT produce sound.

It describes the instrument’s state.

---

## 6. Voice System

Voice is a physical configuration of the instrument.

Defined by:

- mass distribution
- damping behavior
- structural stiffness
- coupling characteristics
- enclosure resonance response

Voice is not a preset or digital mode.

Each voice must exist physically.

---

## 7. Interaction Model

The performer interacts through continuous physical gestures:

- plucking
- sliding
- pressure variation
- optional bowing

There are no discrete trigger zones.

Pitch and tone emerge from continuous interaction with a physical system.

---

## 8. Tuning System

A global tuning lever allows instant switching between reference systems:

- A = 440 Hz (modern reference)
- A = 415 Hz (historical reference)

This is a global harmonic scaling state.

No mechanical retuning is required.

---

## 9. System Hierarchy

1. Mechanical system generates sound
2. Acoustic field carries sound
3. NRE observes and models behavior
4. Electronics optionally interpret state

Causality never flows from electronics to sound.

---

## 10. Instrument Identity

Spectral String is:

- a real acoustic instrument
- a deterministic mechanical system
- a continuous excitation structure
- a reproducible physical design class

---

## 11. Constraint Statement

Any implementation that requires electronics to produce sound is not Spectral String.

Any implementation that removes acoustic autonomy is not Spectral String.

---

## 12. Summary

Spectral String is a physically complete instrument system where:

mechanical excitation → acoustic resonance → musical output

Everything else is optional interpretation.
