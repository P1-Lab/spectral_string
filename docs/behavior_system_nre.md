# Nodal Response Engine (NRE)

The NRE is a structural interpretation system.

It does NOT generate sound.

## Function

It models:

- excitation position
- velocity of interaction
- string tension state
- damping behavior
- harmonic response envelope

## Output

NRE produces state representations of the instrument’s physical behavior.

## Rule

The NRE is descriptive, not causal.

It observes the instrument. It does not create it.
