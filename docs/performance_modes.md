# Performance Modes

Spectral String supports multiple physical interaction regimes. These are not presets — they are **mechanical interpretation layers** that alter how excitation is resolved into spectral state.

---

## 1. Baroque Mode (A=415Hz Reference)

- reduced tension baseline behavior (effective via interpretation layer, not mechanical retensioning)
- slower harmonic stabilization
- increased spectral inertia and drift tolerance

Used for:
- solo performance
- expressive pitch inflection
- bowed or sustained excitation

---

## 2. Modern Ensemble Mode (A=440Hz Reference)

- faster spectral convergence
- tighter harmonic clustering
- increased transient clarity

Used for:
- ensemble integration
- rhythmic articulation
- percussive excitation patterns

---

## 3. Surface Mode

Interaction occurs through coupling with an external or integrated plane.

Effects:
- multi-point excitation via contact propagation
- increased harmonic diffusion
- complex transient scattering across string length

Used for:
- preparatory techniques
- percussive gestures
- distributed excitation (non-point input)

---

## 4. Gravity Interaction Mode (Replaces “Vertical Mode”)

Instrument orientation is treated as a **physical field constraint**, not a stylistic mode.

This is NOT guqin-derived and must not be interpreted as such.

Effects:
- damping response shifts according to gravity vector alignment
- excitation energy distribution becomes asymmetric along string length
- pressure-based input varies with orientation (hand force vectors change mechanically)

Used for:
- exploratory performance states
- bowed or friction-based excitation
- reconfiguration of tactile response behavior under rotation

---

## Mode Switching Mechanism

Mode switching is not MIDI-based or digitally parameterized.

It is implemented via:

- mechanical lever state (reference field selection)
- tension rail geometry adjustment
- bridge angle or coupling stiffness reconfiguration

The system interprets these changes as **state boundary transitions**, not continuous parameter modulation.

---

## Design Principle

Modes do not change the instrument.

They change:

> how the same physical excitation is interpreted as a spectral structure under different mechanical boundary conditions.
