# Performance Modes

Spectral String supports multiple physical interaction regimes. These are not presets — they are **mechanical interpretation layers**.

---

## 1. Baroque Mode (A=415Hz Reference)

- reduced tension baseline
- slower harmonic settling
- higher spectral inertia

Used for:
- solo performance
- microtonal exploration
- bowed sustained gestures

---

## 2. Modern Ensemble Mode (A=440Hz Reference)

- faster spectral convergence
- tighter harmonic clustering
- increased transient responsiveness

Used for:
- ensemble integration
- rhythmic articulation
- percussive excitation

---

## 3. Surface Mode

String is played against a resonant plane.

Effects:
- increased harmonic diffusion
- multi-point excitation
- pseudo-polyphonic behavior

---

## 4. Vertical Mode (Guqin-inspired geometry)

Instrument orientation becomes part of the system.

Effects:
- gravity-aligned damping shifts
- asymmetric harmonic weighting
- bow pressure becomes directional field input

---

## Mode Switching Mechanism

Mode switching is not MIDI-based.
It is implemented via:

- mechanical lever state
- tension rail geometry change
- bridge angle reconfiguration

The system interprets these changes as **state boundary transitions**, not parameter changes.
