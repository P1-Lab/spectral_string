# Voice System (Physical State Architecture)

Voice families define physical configurations of the instrument.

They are not digital presets.

## Voice is defined by:

- mass distribution
- damping characteristics
- structural stiffness
- coupling behavior
- enclosure resonance interaction

## Voice is NOT:

- EQ curves
- DSP presets
- software modes
- signal processing profiles

## Principle

Each voice must be physically reproducible without electronics.

Voice = physical state of the instrument.
