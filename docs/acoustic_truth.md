# Acoustic Truth Layer

The Spectral String system is an acoustic-first instrument system.

Sound is produced through mechanical energy alone:

- string excitation
- structural resonance of the body
- air coupling into the environment

Electronics are optional and do not generate sound.

## Rule

If all electronics are removed, the instrument must still function musically.

## Electronic Role (Optional Only)

- measurement
- analysis
- interpretation
- spectral mapping

Electronics never define primary sound production.
