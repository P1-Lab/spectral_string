# Tuning System

Spectral String tuning is not equal temperament in the conventional sense. It is a **reference stabilization system** for spectral alignment.

---

## Base References

- Baroque reference: A = 415 Hz
- Modern reference: A = 440 Hz
- Extended reference: A = 432 Hz (optional calibration layer)

---

## Tuning Philosophy

Tuning is not pitch correction.

It is:
> alignment of harmonic convergence zones

Each string defines:
- fundamental anchor
- harmonic attractor field
- overtone drift behavior

---

## Continuous Drift Model

Strings are allowed controlled instability:

- slight detuning is intentional
- micro-shifts create spectral richness
- stability is measured, not enforced

---

## Dual-Reference Lever System

A physical lever can switch:

| Position | System State |
|----------|-------------|
| BAROQUE  | 415Hz field anchor |
| MODERN   | 440Hz field anchor |

Switching does not retune the string physically — it reinterprets the same input through a different spectral mapping function.

---

## Microtonal Expansion

The system supports:
- 31-tone subdivision mapping
- adaptive harmonic stretching
- context-dependent interval weighting
