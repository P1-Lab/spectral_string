# STAGE NODE SPECIFICATION
## Spectral String Instrument System

---

## 1. ROLE DEFINITION

The Stage Node is a **projection and ensemble authority instrument**.

It is designed for:

- live performance
- spatial projection
- ensemble anchoring
- high-energy acoustic environments

It emphasizes **presence over precision granularity**.

---

## 2. PHYSICAL DIMENSIONS

| Parameter | Value |
|----------|------|
| Total Length | 40 in (1016 mm) |
| String Span | 36 in (914 mm) |
| Body Width | 3.5 in (89 mm) |
| Body Depth | 1.25 in (32 mm) |

---

## 3. STRUCTURAL CHARACTERISTICS

- High resonance volume
- Extended sustain envelope
- Increased air coupling
- Strong low-frequency reinforcement
