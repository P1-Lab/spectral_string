# MINI NODE SPECIFICATION
## Spectral String Instrument System

---

## 1. ROLE DEFINITION

The Mini Node is a **precision analytical instrument** optimized for:

- high-resolution excitation sensing
- tight spectral control
- fast transient response
- calibration and compositional study

It prioritizes **behavioral clarity over acoustic bloom**.

---

## 2. PHYSICAL DIMENSIONS

| Parameter | Value |
|----------|------|
| Total Length | 22 in (559 mm) |
| String Span | 20 in (508 mm) |
| Body Width | 3.25 in (83 mm) |
| Body Depth | 1.0 in (25 mm) |

---

## 3. STRUCTURAL CHARACTERISTICS

- High stiffness-to-mass ratio
- Reduced acoustic cavity volume
- Fast decay envelope by design
- Minimal internal resonance buildup

---

## 4. ELECTRONICS CAVITY

| Parameter | Value |
|----------|------|
| Length | 5.0 in (127 mm) |
| Width | 2.0 in (51 mm) |
| Depth | 0.65 in (16.5 mm) |

### Constraints
- PCB mechanically constrained on all edges
- Maximum standoff: 0.12 in
- Shielding mandatory (high sensitivity regime)

## 5. BEHAVIORAL PROFILE

- Short sustain
- High articulation clarity
- Reduced harmonic diffusion
- Fast spectral convergence

## 6. USE CASE
