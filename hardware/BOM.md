# Bill of Materials (BOM)

## 1. Structural Frame

- Maple beam (primary inertial body — reference frame material)
- Ash (optional variant — increased mechanical brightness / stiffness response)
- Carbon composite (optional variant — maximum structural stability, minimal coloration)

- Stainless steel mounting rails (string anchor interface backbone)

---

## 2. String System (Substrate-Class A/B/C Compatible)

### Low-Entropy Substrate Set (Steel-Class)
- phosphor bronze strings (standard set)
- stainless steel strings (high stability / precision articulation mode)

### Controlled-Instability Substrate Set (Hybrid Gut-Class)
- synthetic-core / metal-wound hybrid strings (Passione-class behavior family)

### High-Entropy Substrate Set (Gut-Class Optional Expansion)
- gut or gut-derived strings (untreated or lightly stabilized variants)

> Note: substrate selection defines excitation behavior, not instrument identity.

---

## 3. Bridge Assembly

- hardened steel bridge anchor (non-resonant coupling node)
- micro-adjust tail tension coupling (mechanical compensation only)
- rigid, non-flexural mounting interface

---

## 4. Lever System (Spectral Reference Switch)

- detented mechanical switch (binary or dual-state minimum)
- hardened pivot axle (vibration-resistant)
- mechanically isolated mount (no string tension interaction)

> Function: spectral reference frame switching (e.g., 415 Hz / 440 Hz)  
> Not a tuning mechanism.

---

## 5. Sensors (Non-Invasive Field Sampling Layer)

- capacitive linear strip sensor (string position field mapping)
- force-sensitive resistors (bridge excitation envelope detection)
- MEMS accelerometer (global structural vibration tracking, optional)

> Constraint: sensors must remain mechanically neutral to string dynamics.

---

## 6. Hardware Isolation Layer

- rubberized isolation mounts (non-compliant damping zones)
- mechanical decoupling layers between sensing and structural frame
- vibration isolation pads (localized, non-global damping only)

---

## 7. Fasteners

- stainless steel machine screws (M2–M4 range)
- torque-rated fastening points
- lock washers or thread-lock equivalent (non-adhesive preferred)

---

## 8. Electrical (Non-Critical Signal Path)

- high impedance buffer stage (piezo / contact transduction front-end)
- shielded analog routing (noise isolation only)
- isolated ground plane architecture (no shared structural grounding with frame)

> Electrical system is interpretive, not generative.

---

## 9. Prohibited Materials / Design Constraints

- plastic structural beams (mechanical deformation risk)
- low-mass hollow resonant frames (uncontrolled coloration)
- flexible PCB used as structural component
- any active mechanical tuning systems (servo, motor, continuous retensioning)

---

## 10. Substrate Compliance Rule

All string materials must conform to one of the following behavior classes:

- **Low-Entropy (Steel-Class):** deterministic, stable, linear response
- **Controlled-Instability (Hybrid Gut-Class):** bounded nonlinear response
- **High-Entropy (Gut-Class):** thermodynamically sensitive response

The instrument remains valid across all classes provided:
> excitation remains continuous and mapping remains deterministic at the NRE layer
