# Lever Tuning System

## 1. Functional Definition

The lever system does NOT tune strings.

It switches the **interpretive reference frame** used by the spectral engine.

---

## 2. States

### BAROQUE
- A = 415 Hz reference anchor
- spectral mapping widened
- harmonic weighting softened

### MODERN
- A = 440 Hz reference anchor
- tighter harmonic clustering
- faster transient stabilization

---

## 3. Mechanical Implementation

The lever must:

- be physically detented (no analog drift state)
- produce binary or dual-lock engagement
- have zero effect on string tension
- act only on signal interpretation layer

---

## 4. Required Behavior

Switching states must:

- be instantaneous (<10 ms perceived)
- not produce audible mechanical noise in output signal path
- remain stable under vibration

---

## 5. Forbidden Designs

- motorized tuning heads
- continuously variable potentiometer-style tuning
- friction-only ambiguous state switching

Ambiguity is not allowed.

---

## 6. Cognitive Model

The performer should perceive:

> “the same string, reinterpreted differently”

not

> “the string has been retuned”
