# String Tension System

## 1. Design Principle

String tension is not a setup parameter.
It is a **continuous expressive axis**.

Tension defines:
- spectral density
- attack transient shape
- harmonic convergence stability

---

## 2. Tension Bands

### Low Tension (40–60 N)
- high lateral displacement
- strong pitch drift under excitation
- suited for glissandi and bowed instability

### Nominal Tension (60–90 N)
- balanced harmonic response
- stable overtone structure
- default performance mode

### High Tension (90–120 N)
- reduced excursion
- sharp transient response
- percussive articulation dominance

---

## 3. Dual-String Interaction (if present)

When two strings are installed:

- tension asymmetry is allowed
- cross-beating is structurally meaningful
- harmonic locking is a valid performance state

---

## 4. Stability Rule

The system must tolerate:

- ±5% tension variation without recalibration
- thermal drift without compensation systems

If compensation is required, it must be mechanical, not electronic.

---

## 5. Non-Allowed Systems

- automatic tuning motors
- servo tension correction
- MIDI-driven retensioning

These violate the physical integrity model.
