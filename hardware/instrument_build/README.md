# Spectral String — Instrument Constitution

## Status
Active Acoustic Instrument System (not a controller, not a synth framework)

---

## 1. Identity

Spectral String is a physically complete acoustic instrument family.

It produces sound through:
- mechanical string excitation
- structural resonance of the instrument body
- deterministic physical coupling between components

No electronics are required for sound production.

---

## 2. Core Principle

The instrument exists before computation.

Sound is generated physically.  
Computation is optional interpretation.

---

## 3. Acoustic Independence

If all electronics are removed:

- the instrument remains playable
- pitch remains mechanically defined
- timbre remains structurally stable
- performance remains musically valid

Electronics do not generate sound. They observe and extend it.

---

## 4. Instrument Models

- Monos: single-string continuous instrument
- Duos: dual-string harmonic interaction instrument

Both share identical acoustic architecture principles.

---

## 5. Design Philosophy

- Mechanical behavior defines musical identity
- Electrical systems do not override physical causality
- Performance is gesture-to-structure interaction, not event triggering

---

## 6. System Scope

This repository defines:

- physical instrument construction
- mechanical excitation behavior
- optional spectral interpretation systems
- calibration and measurement frameworks

---

## 7. Constraint

This is not:

- a MIDI controller
- a digital synthesizer
- a sample playback system

It is a **physical acoustic instrument system with optional computational extension**.
