# Spectral String — Mechanical Specification (v1.1)

---

## 1. Instrument Class

Spectral String is a **continuous excitation acoustic instrument** based on tensioned string physics with embedded sensing and deterministic spectral interpretation.

It is not a controller, not a keyboard, and not a trigger surface.

The mechanical system must remain fully playable without power at all times.

---

## 2. Structural Layout

### Core Architecture

- 1 or 2 tensioned strings (Monos / Duos configurations)
- rigid beam body (non-resonant primary inertial frame)
- distributed sensing along string path
- optional coupling surface beneath string plane

---

## 3. String System (Substrate-Compatible Design)

The instrument supports multiple physical substrate classes:

### A. Low-Entropy Substrate (Steel-Class)
- stainless steel strings (precision / stability)
- phosphor bronze strings (balanced harmonic response)

### B. Controlled-Instability Substrate (Hybrid Gut-Class)
- synthetic-core / wound hybrid strings (Passione-class behavior family)

### C. High-Entropy Substrate (Gut-Class Optional)
- gut or gut-derived strings (untreated or minimally stabilized variants)

---

### Tension Range (Mechanical Capability Envelope)

- Low: 40–60 N (high displacement / expressive deformation)
- Nominal: 60–90 N (balanced structural response)
- High: 90–120 N (maximum articulation clarity)

> Note: actual perceptual stability depends on substrate class, not only tension.

---

### Mounting

- fixed bridge anchor (non-moving structural reference)
- floating tail anchor with micro-adjust compensation (mechanical only)
- no active or motorized tension systems permitted

---

## 4. Body Construction

### Materials (ranked)

1. Maple (preferred)
   - balanced stiffness and tactile feedback
2. Ash
   - increased mechanical brightness and attack response
3. Carbon composite
   - maximum structural stability, minimal coloration

---

### Structural Rule

The body must behave as:

> a reference inertial frame, not a resonant sound-shaping cavity

Resonance is permitted only if explicitly engineered and controlled.

---

## 5. Playing Surfaces

### Permitted Interaction Domains

- string (primary excitation medium)
- top beam surface (secondary tactile control field)
- side edges (gesture articulation vectors)
- underside coupling surface (optional interaction plane)

No discrete or fixed interaction zones are permitted.

Interaction is spatially continuous.

---

## 6. Sensor Integration

### Allowed Sensor Types

- linear capacitive strip (string position field mapping)
- pressure sensing elements (force envelope detection)
- optional MEMS accelerometer (global structural motion field)

---

### Placement Constraint

Sensors must remain mechanically neutral:

- no alteration of string stiffness
- no damping of vibrational modes
- no mass loading that changes resonance behavior

---

## 7. Lever System (Spectral Reference Switch)

### Function

Switches global spectral reference frame:

- A = 440 Hz (modern ensemble reference)
- A = 415 Hz (historical / solo reference)

---

### Implementation Requirements

- binary or detented mechanical switch
- no physical string retensioning
- no mechanical pitch alteration
- operates only at spectral interpretation layer (NRE domain)

---

## 8. Orientation Modes (Corrected Model)

Orientation does NOT define instrument identity.

It defines **gravity interaction geometry**.

### Planar Interaction Mode
- instrument aligned perpendicular to gravity vector
- stable pressure-based articulation
- maximum positional resolution (guqin-class interaction behavior, but not stylistically referenced)

### Vector Interaction Mode
- instrument rotated relative to gravity vector
- damping becomes directionally asymmetric
- excitation response varies with orientation and force vector

> Note: behavior differences arise from physics, not stylistic emulation.

---

## 9. Design Constraint: Acoustic Integrity

The instrument must remain:

- fully playable with electronics OFF
- responsive without amplification
- mechanically stable under dynamic excitation

If power is required for basic acoustic behavior, the system is invalid.

---

## 10. Mass Distribution

- central beam: high stiffness / low damping (reference frame stability)
- string anchors: high rigidity coupling points
- sensor regions: neutral mass loading (no vibrational sinks or dampers)

No component may unintentionally create sympathetic resonance unless explicitly designed.

---

## 11. Aesthetic Constraint (Functional, not decorative)

The instrument must visually communicate:

- tension
- structural causality
- mechanical precision
- physical force pathways

It must not resemble consumer MIDI or interface hardware.

---

## 12. Certification Rule

A valid Spectral String build must:

- produce consistent pitch mapping across full string length
- maintain stable transient response under repeated excitation
- preserve reference switching (415/440) without mechanical drift
- maintain functional playability across orientation modes (with expected physical response differences accounted for)
