# Mechanical System Definition

The instrument is a physically coupled acoustic system.

## Core Components

- tensioned string system
- bridge and nut boundary structure
- resonant body (wood or composite enclosure)
- human-to-string excitation interface

## Principle

All sound originates from:

mechanical energy transfer → string vibration → body resonance → air radiation

## Constraint

No electronic or computational system may define or generate primary acoustic output.
